Úprava testů

Přidali jsme test s fragmentem například https://github.com/encukou/freezeyt/#usage obsahuje fragment #usage - je to odkaz na konkrétní místo na dané stránce, ale náš freezer to může ignorovat, protože to zajímá prohlížeč. Přidat test s ftp -> nyní freezer bere jen http a https schéma Rozebrali jsme strukturu url adresy a to, co nás v ní zajímá

  • schéma - bereme jen http a https
  • netloc - zajímá nás - náš server to musí být
  • path - zajímá nás
  • parametry - nepoužívají se
  • query (dotazy) - nás nezajímají, to funguje jen v aplikacích
  • fragmenty - zajímají prohlížeč a ne freezer

Z proměnné url jsme udělali url_path - aby to bylo výstižnější
